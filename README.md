# Weather NGS

</div>

Учебный проект, написанный на языке программирования **Kotlin** с использованием архитектуры **MVVM (Model-View-ViewModel)** с **Single Activity** и UI на **Jetpack Compose**.

Android-приложение для отображения текущей погоды и прогноза на 10 дней для выбранного города из списка доступных.
В качестве источника данных используется [НГС API](https://pogoda.ngs.ru/).

<img src=".github/previews/weatherScreen.gif" width="300" align="right" style="margin-left: 50px;" alt="Демонстрация работы приложения">

## Функциональность
- **Текущая погода** – отображение температуры, влажности, облачности и осадков
- **Прогноз на несколько дней** – детальная информация по дням
- **Поиск городов** – возможность выбора из списка доступных городов
- **Оффлайн-режим** – работа без интернета с использованием кешированных данных
- **Автообновление** – актуальные данные с сервера

## Стек технологий и библиотек
- Минимальный **Android SDK** – 26
- На основе **Kotlin**, **Coroutines** + **Flow**
- **Jetpack**
    - **Compose** – Современный декларативный UI-фреймворк
    - **Navigation Compose** – Навигация между composable-экранами
    - **ViewModel** – Хранение UI-данных с учетом жизненного цикла
    - **Room** – ORM библиотека для работы с локальной БД
    - **DataStore** – Кеширование данных последнего отображаемого города 
- **Архитектура**
    - **MVVM (Model-View-ViewModel)**
    - **Clean Architecture** с разделением на слои
    - **Repository Pattern**
- **Retrofit 2** – REST API клиент
- **Koin** – Легковесный DI-фреймворк
- **Kotlin Serialization** – JSON сериализация
- **Material Design 3** – Современная дизайн-система с поддержкой динамических цветов
